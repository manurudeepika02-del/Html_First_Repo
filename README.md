## 📖 Description
Explore Nature is a beginner-friendly web project created using pure HTML5.
The project demonstrates essential HTML elements required to design a complete webpage including media, lists, forms, and iframe integration.

## ✨ Features
- Header & Footer sections
- Text formatting tags
- Images and media embedding
- Ordered & Unordered lists
- YouTube video integration
- Audio player support
- Google Maps embedding
- Contact form
